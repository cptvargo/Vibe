# Vibe Artist Images

Drop artist photos here. Named exactly as they appear in your Jellyfin library.

## Supported formats
- .jpg
- .png  
- .webp

## Examples
```
GodFearin.jpg
Andy Mineo.jpg
Lecrae.png
Elevation Worship.jpg
Bethel Music.jpg
Thaddeus Brown.jpg
EmanuelDaProphet.jpg
```

## Tips
- Any size works — Vibe crops to fit the hero
- Portrait photos work best (taller than wide)
- High res preferred — at least 500x500px
- Vibe checks this folder FIRST before trying Jellyfin or any external source
